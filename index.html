Research interests

  • Analysis of machine learning models

    Is the process of understanding, diagnosing, and refining machine learning models with the help of interactive visualization, that leads users to improve the performance of their built models and thus to efficiently solve real-world artificial intelligence and data mining problems.

  • Parallel genetic algorithms

    The parallel genetic algorithms (PGAs) is a prototype of a new kind of a distributed algorithms which are used to solve complex optimization problems, where multiple genetic algorithms are used to solve a single task. All these algorithms try to solve the same task and after theyâ€™ve completed their job, the best individual of every algorithm is selected, then the best of them is selected, which is the solution to a problem. These genetic algorithms do not depend on each other, as a result, they can run in parallel, taking advantage of a multicore CPU. Parallel genetic algorithms are useful especially when a task is difficult to solve or an individual is a complex entity.

  • Computer vision-based object detection

    Computer vision is regarded with the theory behind artificial systems that elicit information from images. The image data can take many forms, such as views from multiple cameras, video sequences, multi-dimensional data from a 3D scanner device, or medical scanning device. Examples of computer vision include scene reconstruction, Image restoration , object detection, video tracking, 3D pose estimation, motion estimation, and 3D scene modeling.

  • Parallelsim of deep learning models

    This area focuses on scaling up machine learning, data mining, especially for massive datasets. In this area, several programming environments and frameworks have been built for dealing with multithreading (e.g., OpenMP), multiprocessing (e.g mpi), many-cores(e.g., OpenCL, CUDA, OpenACC). Parallel techniques are used to scale up AI algorithms, using one of these frameworks, and run them on clusters, many-cores, or specialized accelerators.
